OneBot API

properties/functions in red are deprecated and/or not meant for use.

channels and parameters are shown in blue

replacements/to use instead are shown in green

shard is deprecated.


Default API (--api [cmd], channel: api_main)

ping: Shows the ping of the api servers.

parameters: empty, all, mains, backplate, fallback, shard.

replacement: --status


user: Request the properties of yourself.

parameters: id, username, tag


properties: Request API properties (main channel only)

parameters: help, list, version

replacement: --api apiprops


apiprops: API Properties, status and channel

parameters: mains, backplate, fallback, shard.


Backplate API (--api-b[cmd], channel: api_backplate)

The Backplate is using a command handler, rather than one file with else if.

ping: Shows the ping of the backplate api server.

replacement: --status


properties: Request API Properties (backplate only)

parameters: list

replacement: none


user: Request the properties of yourself.

parameters: id, username, tag


Fallback API (--api-f[cmd], channel: api_main (backplate-fallback))

The fallback API is using a command handler, rather than one file with else if.

The fallback API uses an older version of the api. Please keep that in mind when using it.

All command are currently the same as the backplate. However, the properties are fallback rather than backplate.


shard (channel: shard_element)

Shard is deprecated.
