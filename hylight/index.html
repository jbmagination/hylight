hylight - the simplest programming language ever.

v1.5 Released! (stable)


v1.5 now supports hss!

hylight an extremely simplified version of HTML - you are quite literally writing in plain english.
the hylight converter is written purely in javascript.

lets get straight to the point - you want to learn how to use it.

any functions (eg h1, p, image) in red is buggy or deprecated

first you need to install a build. by using one the download buttons above.

after this, install a node.js build from node.js.org. minimum v11.12.0.

first create a file called index.hyl

formatting

hylight is formatted in the following way:


tag data/string
tag data/string

IMPORTANT - "ENTER" IS REQUIRED BETWEEN TAGS (SHOWN ABOVE)

for example, if you wanted a heading 1 saying "hello there", you would do the following:


h1 hello there

IMPORTANT - HOW TO CONVERT .HYL TO .HTML

Run convert.bat and then open index.html

Headings (v1.0)

Headings look like this:

hello

hello

hello

hello

hello
hello

h1 hello

replace 1 with anything 1-6

Paragraphs (v1.0)

Paragraphs look like this:

hello


p hello

Images (v1.0)


image src

Replace src with a link or a directory, eg ./test.png

subscript (v1.0) and supscript


sub hello
sup hello

Break Statements (v1.0)

Break statements can be done in two different ways


br
break

Audio (v1.2)

Audio is based on what type you are using - MP3 or OGG

MP3:


audio(mp3) ./name.mp3

OGG:


audio(ogg) ./name.ogg

Right to left text (v1.2)


bdo [text]

iFrame Support (v1.2)

iFrames are pretty bugged right now but sometimes they do work

You need to specify a link including the http type.


iframe https://youtube.com

Video (v1.3)

Video is currently only supported in mp4 format.


video ./name.mp4

International Language (v1.3)

International Character Support


bdi [text]
