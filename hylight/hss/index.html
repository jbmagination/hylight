hss - simple css made for hylight projects.

v0.1 Released! (stable)


the hss converter is written purely in javascript.

im going to assume you have read the hylight documentation - so i wont bore you

IMPORTANT - HOW TO CONVERT .HSS TO .CSS

Open the node.js command prompt, use cd to go to your directory of hss then do node convert.js, you can now use styles.css
